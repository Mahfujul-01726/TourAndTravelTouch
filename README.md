## рҹҺҜ Mission

**Tour And Travel Touch** bridges travelers with Bangladesh's most extraordinary destinations. Built as a full-stack booking platform with a **vanilla PHP + MySQL** backend and an **immersive, animated frontend**, it turns trip planning into an experience вҖ” not just a transaction.

From the world's largest mangrove forest to the pristine beaches of the Bay of Bengal, every corner of this platform is crafted to inspire wanderlust.

| Feature | Implementation | Highlights |
|---------|:-------------:|:----------:|
| **Booking Engine** | PHP + MySQL via `mysqli` | Prepared statements, date validation, traveler count limits |
| **Instant Search** | PHP search handler | Masked input fields, SQL-injection safe, real-time results |
| **User Registration** | PHP auth handler | Bcrypt hashing (cost 12), email uniqueness, CSRF-guarded |

## рҹҡҖ Quick Start

Everything runs on **InfinityFree** вҖ” one domain, no cross-origin issues.

### Step 1: Create InfinityFree Account

1. Go to **[infinityfree.com](https://infinityfree.com)** вҶ’ **Sign Up**
2. Confirm your email вҶ’ Log in to the control panel

### Step 2: Create Database

1. Go to **MySQL Databases** вҶ’ **Create Database**
2. Copy the credentials (they look like this):
   ```
   Hostname: sqlXXX.infinityfree.com
   Database: if0_XXXXX_databasename
   Username: if0_XXXXX
   Password: (shown once вҖ” save it)
   ```

### Step 3: Import Schema

1. Open **phpMyAdmin** вҶ’ select your database
2. Click **Import** вҶ’ choose `database/schema.sql` вҶ’ **Go**

### Step 4: Configure

Open **`backend/config/database.php`** and paste your credentials:

```php
define('DB_HOST', 'sql305.infinityfree.com');
define('DB_NAME', 'if0_41936508_firstsql');
define('DB_USER', 'if0_41936508');
define('DB_PASS', '3a4jNkVeAq');
```

> If you re-import the schema, also run this SQL in phpMyAdmin to create the admin table:
> ```sql
> CREATE TABLE IF NOT EXISTS admins (
>     id INT AUTO_INCREMENT PRIMARY KEY,
>     username VARCHAR(100) NOT NULL UNIQUE,
>     password_hash VARCHAR(255) NOT NULL,
>     created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
> ) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;
> ```

### Step 5: Upload Everything

1. File Manager вҶ’ go into `htdocs/` вҶ’ delete default files
2. **Upload the entire project** вҖ” all files and folders
3. Wait for upload to complete

### Step 6: Launch рҹҺү

Visit your InfinityFree domain вҖ” everything works on one domain:

```
https://tourandtraveltouch.great-site.net
```
