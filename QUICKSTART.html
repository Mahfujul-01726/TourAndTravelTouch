🎨 Quick Customization

1. Change Brand Colors

Edit the color variables in style.css:

:root { --primary-color: #FF6B35; /* Orange */ --secondary-color: #004E89; /* Blue */ --accent-color: #1AC8ED; /* Cyan */ }

2. Update Package Information

Find the "Featured Packages" section in index.html and update:

<h4>Your Package Name</h4> <p class="package-description">Your description</p> <div class="price">From <strong>৳ Your Price</strong></div>

3. Add Your Images

Replace image URLs in the code with your own:

style="background-image: url('YOUR_IMAGE_URL_HERE');"

4. Update Contact Information

Find the Contact Section and update:

+880 1700-000000 → Your phone number info@tourtraveltouch.com → Your email Dhaka, Bangladesh → Your location

💡 Pro Tips

Tip 1: Use Google Fonts

Change fonts in style.css or add new ones from Google Fonts for a unique look.

Tip 2: Add More Destinations

In script.js, modify the destinations array to show your featured places:

const destinations = ['Place1', 'Place2', 'Place3', 'Place4'];

Tip 3: Custom Animations

Add custom CSS animations in style.css for unique effects.

Tip 4: Social Media Integration

Update social links in the footer to point to your accounts.

🆘 Need Help?

Common Issues

  • Pages not loading? → Check file paths are correct
  • Styles not applying? → Clear browser cache (Ctrl+Shift+Delete)
  • Images broken? → Verify image URLs are accessible
  • Forms not working? → Check PHP configuration and database
